❓ 자주 묻는 질문
Q. 내 데이터는 안전한가요?
네. 각자 본인 Slack 계정으로만 OAuth 연동되고, 토큰은 사용자별로 격리돼요. 다른 사람 메시지 못 봐요.
Q. 여러 대화창 동시에 열 수 있나요?
Chrome에서는 가능 (카톡 PC앱처럼 별도 창). 다른 브라우저는 한 번에 하나씩.
Q. 알림이 느려요
업무시간(9-12, 13-18) 3초, 점심 10초, 저녁 5초, 야간 60초 간격으로 확인해요. 가볍게 쓰려고 주기를 조정했어요.
Q. 이미 공식 Slack 앱 쓰고 있는데?
병행 가능해요. 이 대시보드는 가볍게 메시지만 확인할 때 편해요.

💾 바탕화면에 설치하기
매번 브라우저 주소 입력 없이, 바탕화면에서 바로 대시보드 열기!
⭐ 옵션 A: Chrome 앱으로 설치 (추천)
원클릭 설치 · 시작 메뉴 + (선택) 바탕화면 자동 등록 ·
독립 창 (카톡 PC 앱처럼 브라우저 UI 없음)
⚠️ 지금 브라우저에서 PWA 설치를 지원하지 않아요.
Chrome으로 접속해주세요. 또는 아래 옵션 B 사용.
📌 설치 후 바탕화면에 안 보이면:
① 시작 메뉴 → "Slack" 검색 → 마우스 우클릭
② "파일 위치 열기" 클릭
③ 나타난 아이콘을 바탕화면으로 드래그
또는 → 작업표시줄에 "고정" 가능
🛠️ 옵션 B: Windows 바로가기 다운로드
.bat 파일 다운로드 → 바탕화면으로 드래그 → 더블클릭하면
무조건 Chrome 앱 모드로 열림 (기본 브라우저 무시)
💡 Windows가 "보호" 경고 표시할 수 있어요 → "실행" 또는 "추가 정보 → 실행" 누르세요
🍎 Mac 사용자
Chrome에서 ⋮ 메뉴 → "도구 더보기" → "바로가기 만들기..."
또는 Safari에서 공유 → Dock에 추가
